# Node.js + Koa2 + Typescript 构建服务端应用

当前描述文件不做详细介绍，请移步到[掘金介绍](https://juejin.cn/post/6844903922767757320/)

## 安装依赖

```
npm install
```

## 运行并开启代码热更新

```
npm run dev
```

## 直接编译输出到`dist`目录并运行

```
npm run build
```
